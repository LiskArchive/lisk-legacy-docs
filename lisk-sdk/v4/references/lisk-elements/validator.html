Validator

Table of Contents
  • Installation
  • Update

@liskhq/lisk-validator is a wrapper of validation packages. It includes custom validations utilities related to the Lisk Protocol.

Installation

How to perform the installation:

npm install @liskhq/lisk-validator@0.4.0

Update

How to perform an upgrade:

npm update @liskhq/lisk-validator
